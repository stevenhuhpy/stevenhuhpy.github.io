声明成员函数为 const 意味着该函数被禁止修改调用它的对象，包括：

  • 无法修改任何非静态的数据成员
  • 调用任何非静态成员函数。

关键字 const 必须同时标记到声明和实现。

非类成员函数无法声明为 const。
